# Module 25: Capstone Projects and Portfolio for Interview Success

## Learning Objectives
- Build interview-ready evidence of hands-on capability.
- Present project outcomes in senior-level language.

## Capstone 1: Enterprise Image Hardening Program
Goal:
- Create a standard hardening baseline and apply it across multiple sample services.

Deliverables:
- Hardening standard document.
- Before/after scan comparison.
- Image size and risk reduction metrics.

## Capstone 2: Continuous CVE Remediation Pipeline
Goal:
- Build automated pipeline with scan gates, exception handling, and audit trail.

Deliverables:
- CI workflow file.
- Policy gate rules.
- Exception template with expiry.

## Capstone 3: Supply Chain Trust Implementation
Goal:
- Add SBOM generation and image signing verification flow.

Deliverables:
- SBOM artifacts for each build.
- Signature verification evidence.
- Deployment trust policy summary.

## Capstone 4: Incident Drill Simulation
Goal:
- Simulate response to a critical CVE affecting multiple services.

Deliverables:
- Incident timeline.
- Containment and remediation plan.
- Post-incident improvement actions.

## Portfolio Presentation Template
For each project show:
1. Problem and risk context.
2. Technical implementation.
3. Validation approach.
4. Results and metrics.
5. Lessons learned.

## Interview Use
In interviews, pick one capstone and explain:
- Why the design choices were made.
- What tradeoffs were accepted.
- What business impact improved.
